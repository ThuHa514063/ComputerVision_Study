# Computer Vision

## OpenCV

### Thư viện ```cv2```

**Cài đặt:** gõ lệnh để tải xuống thư viện cv2 trong terminal. Lệnh:
```
python -m pit install opencv-python
```

**Kiểm tra version:**
```
python -m pip show opencv-python
```

=> Kết quả:
```
Name: opencv-python
Version: 4.13.0.90
```
